# Control de clientes

Sitio adaptativo para registrar clientes que piden catalogo.

## Uso rapido

1. Abre `index.html` en el navegador.
2. Agrega un numero de WhatsApp y observaciones.
3. Cambia los estados con los botones de Si/No.
4. Usa el buscador, filtros y orden para encontrar registros.
5. Exporta o importa CSV desde la seccion de herramientas.

Los registros se guardan en el navegador usando LocalStorage.
